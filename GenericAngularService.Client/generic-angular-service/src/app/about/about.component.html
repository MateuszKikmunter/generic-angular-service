- About Angular Company Manager -

I created this project to improve my Angular skills by writing a small application with some CRUD operations. Main goal was to create a generic and reusable service to fetch data from web api and then view it on a webpage.

Such service is extremely useful as it greatly reduces amount of code which has to be written to get data from an external data source. Additionally it's very easy to unit test and flexible when it comes to extend it with additional features.

Technologies Used:

  • Angular 6
  • Angular Datatables
  • Angular FontAwesome
  • ASP.NET Core Web Api
  • Bootstrap 4
  • Bootswatch
  • Dynamic Linq
  • Entity Framework Core
  • Jasmine
  • Karma
  • NgBootstrap
  • NUnit
  • SQL Server
  • Toastr
